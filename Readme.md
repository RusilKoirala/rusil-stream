# Rusil Stream

Rusil stream is a ecosystem of mobile, webapp of streaming movies and tvshows as a educational project usin tMdb and vidsrc api's


## Folder structure 

`/mobile` contains mobile code 
`/web-app` contains web-app and api 


## Project Flow 

The project flows like first web-app is started in vercel or any hosting platform it provides all the api 

The api call in that api made through mobile app (expo app )
